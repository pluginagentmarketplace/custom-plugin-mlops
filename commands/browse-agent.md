# /browse-agent

Explore all 7 specialized agents and their capabilities in Ultrathink custom-plugin-mlops.

## What is this?

This command lets you discover and browse through all available agents and their specializations.

## How to Use

```
/browse-agent [agent-name]
```

### Examples

```
/browse-agent
/browse-agent frontend
/browse-agent backend
/browse-agent all
```

## How to Choose Your Agent

### By Technology Stack

- **Frontend-focused** → Frontend & Design Systems
- **API/Backend-focused** → Backend & API Development
- **Mobile Apps** → Mobile Development
- **Data & AI** → Data Science & AI/ML
- **Infrastructure** → DevOps & Cloud Infrastructure
- **Databases** → Database & Data Management
- **System Design** → Software Architecture & Leadership

### By Career Goals

- **Start coding** → Frontend or Backend or Mobile
- **Build AI apps** → Data Science & AI/ML
- **Infrastructure** → DevOps & Cloud Infrastructure
- **Database expert** → Database & Data Management
- **Architect systems** → Software Architecture & Leadership
- **Lead teams** → Software Architecture & Leadership

### By Experience Level

- **Beginner** → Pick any agent, start with fundamentals
- **Intermediate** → Deepen in one domain or expand to multiple
- **Advanced** → Specialize or learn architecture patterns
