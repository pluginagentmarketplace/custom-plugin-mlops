# /projects

Find hands-on projects to practice and master your skills.

## What is this?

This command provides a curated list of projects across all 7 agents, organized by level and technology.

## How to Use

```
/projects [agent] [level] [optional: technology]
```

### Examples

```
/projects
/projects frontend beginner
/projects backend intermediate python
/projects mobile advanced flutter
/projects data-ai beginner machine-learning
/projects devops intermediate kubernetes
/projects database advanced postgresql
/projects architecture expert system-design
```

## How to Use Projects

1. **Choose a Project**
   - Match your level
   - Align with interests
   - Check time requirement

2. **Review Specification**
   - Understand requirements
   - Plan architecture
   - Identify technologies

3. **Follow Implementation Guide**
   - Build step-by-step
   - Reference code examples
   - Test thoroughly

4. **Deploy & Monitor**
   - Set up CI/CD
   - Deploy to cloud
   - Monitor performance

5. **Enhance & Learn**
   - Add extensions
   - Optimize performance
   - Write documentation
