## Key Learning Paths

### SQL & Relational Databases (Foundation)

#### Phase 1: Fundamentals (2-4 weeks)
- Database concepts and terminology
- SQL basics (SELECT, WHERE, ORDER BY)
- Data types and constraints
- Creating tables and schemas

#### Phase 2: Core Operations (2-4 weeks)
- CRUD operations (INSERT, UPDATE, DELETE)
- SELECT queries and filtering
- Sorting and limiting results
- Basic joins (INNER, LEFT, RIGHT)

#### Phase 3: Multi-Table Queries (2-3 weeks)
- INNER, LEFT, RIGHT, FULL OUTER joins
- Subqueries and nested queries
- UNION and set operations
- Table aliasing

#### Phase 4: Data Analysis (3-4 weeks)
- Aggregate functions (COUNT, SUM, AVG, MIN, MAX)
- GROUP BY and HAVING clauses
- Data aggregation and summarization
- Correlation and pivot tables

#### Phase 5: Advanced Queries (2-3 weeks)
- Common Table Expressions (CTEs)
- Window functions (ROW_NUMBER, RANK, DENSE_RANK)
- Partitioning and ordering
- Advanced analytics

#### Phase 6: Performance & Optimization (3-4 weeks)
- Indexing strategies (B-tree, Hash, BRIN)
- Query execution plans (EXPLAIN ANALYZE)
- Query optimization techniques
- Performance tuning

#### Phase 7: Transactions & Control (2-3 weeks)
- ACID properties understanding
- Transaction isolation levels
- Locking mechanisms
- Concurrency control
